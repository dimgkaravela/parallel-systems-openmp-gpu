# Parallel Systems: OpenMP and GPU Programming

C implementations developed for the **Parallel Systems and Programming** course at the University of Ioannina.

This repository contains CPU and GPU parallel programming exercises using OpenMP, CUDA, and OpenMP offloading. The implemented programs focus on parallel prime number computation, Gaussian blur image processing, and GPU device information retrieval.

## Overview

The project is divided into two parts:

1. **OpenMP CPU Parallelism**
   - Parallel computation of prime numbers
   - Gaussian blur image processing using OpenMP loops and tasks

2. **GPU Programming**
   - CUDA device information retrieval
   - Gaussian blur image processing using OpenMP GPU offloading
